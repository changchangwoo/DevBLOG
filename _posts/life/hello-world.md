# 안녕하세요!

Next.js 기반 개발 블로그에 오신 것을 환영합니다.

## 이 블로그에 대해

이 블로그는 다음과 같은 기술 스택으로 구축되었습니다:

- **Next.js 16** - 정적 사이트 생성 (SSG)
- **Markdown** - 포스트 작성 및 관리
- **Tailwind CSS 4** - 스타일링
- **TypeScript** - 타입 안정성

## Markdown 기능

코드 블록도 지원합니다:

```typescript
export default function HelloWorld() {
  return <h1>Hello, World!</h1>;
}
```

리스트도 잘 작동합니다:

- 항목 1
- 항목 2
- 항목 3

**굵은 글씨**와 *기울임 글씨*도 사용할 수 있습니다.

앞으로 좋은 콘텐츠로 찾아뵙겠습니다!
